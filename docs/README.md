# UniApp X UI 文档

这是 UniApp X UI 组件库的文档源文件目录。

## 目录结构

- `.vitepress/` - VitePress 配置文件
- `public/` - 静态资源文件（Logo、图标等）
- `guide/` - 指南文档
- `components/` - 组件文档
- `index.md` - 文档首页

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run docs:dev

# 构建文档
pnpm run docs:build

# 预览构建结果
pnpm run docs:preview
```
