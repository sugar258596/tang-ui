# 搜索功能

UniApp X UI 文档系统提供了强大的本地搜索功能，帮助您快速找到需要的内容。

## 如何使用搜索

### 方式一：使用快捷键

您可以使用以下快捷键快速打开搜索框：

- **Windows/Linux**: `Ctrl + K`
- **Mac**: `Cmd + K`
- **通用**: 按 `/` 键

### 方式二：点击搜索按钮

在页面顶部导航栏，点击"搜索文档"按钮即可打开搜索框。

## 搜索功能特性

### 1. 模糊搜索

搜索支持模糊匹配，即使您输入的关键词不完全准确，也能找到相关内容。

**示例**：
- 搜索 "butn" 可以找到 "Button 按钮"
- 搜索 "输入" 可以找到 "Input 输入框"、"Textarea 多行文本"、"NumberInput 数字输入" 等

### 2. 前缀搜索

搜索支持前缀匹配，输入关键词的开头部分即可找到相关内容。

**示例**：
- 搜索 "Inp" 可以找到 "Input 输入框"
- 搜索 "按" 可以找到 "Button 按钮"

### 3. 智能权重

搜索结果按照相关性排序，标题匹配的结果会优先显示：

- **标题匹配** - 权重最高（4倍）
- **正文匹配** - 权重中等（2倍）
- **其他匹配** - 权重较低（1倍）

### 4. 实时搜索

输入关键词后，搜索结果会实时更新，无需等待。

### 5. 键盘导航

在搜索框中，您可以使用键盘快速导航：

- `↑` / `↓` - 在搜索结果中上下移动
- `Enter` - 打开选中的搜索结果
- `Esc` - 关闭搜索框

## 搜索技巧

### 搜索组件

直接输入组件名称或功能关键词：

- 搜索 "Button" 或 "按钮" - 找到按钮组件
- 搜索 "Input" 或 "输入框" - 找到输入框组件
- 搜索 "表单" - 找到所有表单相关组件

### 搜索功能

输入功能相关的关键词：

- 搜索 "v-model" - 找到所有支持双向绑定的组件
- 搜索 "懒加载" - 找到支持懒加载的组件
- 搜索 "主题" - 找到主题相关的文档

### 搜索 API

输入 API 相关的关键词：

- 搜索 "Props" - 找到包含属性说明的文档
- 搜索 "Events" - 找到包含事件说明的文档
- 搜索 "Slots" - 找到包含插槽说明的文档

## 搜索范围

本地搜索会索引以下内容：

- ✅ 所有页面标题
- ✅ 所有页面正文内容
- ✅ 代码示例中的注释
- ✅ API 表格中的说明

## 常见问题

### 为什么搜索不到某些内容？

1. **内容尚未创建** - 部分组件文档可能还在编写中
2. **关键词不匹配** - 尝试使用不同的关键词或更通用的词汇
3. **搜索索引未更新** - 刷新页面重新加载搜索索引

### 如何提高搜索准确性？

1. **使用准确的关键词** - 使用组件名称或功能名称
2. **使用中英文混合** - 如 "Button 按钮"
3. **使用简短的关键词** - 避免输入完整的句子

### 搜索快捷键不起作用？

1. 确保焦点在文档页面上（不在输入框中）
2. 检查浏览器是否拦截了快捷键
3. 尝试使用 `/` 键作为替代方案

## 技术实现

UniApp X UI 文档使用 VitePress 内置的本地搜索功能：

- **搜索引擎**: MiniSearch
- **索引方式**: 构建时生成静态索引
- **搜索算法**: 模糊匹配 + 前缀匹配
- **性能**: 纯客户端搜索，无需服务器
